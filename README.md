# Hide Blocked Users

### Hide Blocked Users in Reddit / Discord

---

## English

A browser extension that hides comments from users you have blocked on Reddit and Discord.

- **Reddit**: Hides comments by blocked users (and deleted accounts) in post threads.
- **Discord**: Hides messages from blocked users within servers.

> **Note:** This extension cannot hide posts or direct messages (DMs) from blocked users — it only hides their comments/messages in threads and server channels.

---

## 日本語

RedditやDiscordでブロックしたユーザーのコメントを非表示にするブラウザ拡張機能です。

- **Reddit**：投稿に対するブロック済みユーザーのコメントを非表示にします。削除済みアカウント（deleted user）のコメントも非表示になります。
- **Discord**：サーバー内でのブロック済みユーザーのメッセージを非表示にします。

> **注意：** ブロックしたユーザー本人の投稿やDM（ダイレクトメッセージ）は非表示にできません。スレッドやサーバーチャンネル内のコメント・メッセージのみが対象です。
